Online Shop

This project was built as a part of the recruitment process. I was supplied with a design in figma and a graphical server. I think it was a really cool project to work on, and I have learned quite a lot from it, Especially on graphQl as I never worked with it before, but also on how to implement redux. Design was only supplied for desctop screen, and responsivenes was not required, so the website is not responsive.

Here you can see a product page, where the customer needs to select the attributes like: Size, color or disc space for example depending on the type of item. Once all attributes are selected item can be added to cart.

Users can display mini cart by clicking on cart icon, where they will find a gallery for each item, and all other information such as selected color and size. By clicking on dollar sign users can select the currency they would like to pay in.

On the cart page users can display all the information for each product like in mini cart, but they can also see all avalibal images, and see a better breakdown of price, for example how much tax they would pay.

Currently I don't have a live website for this project, but I would like to invite you to visit github page
